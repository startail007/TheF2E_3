## 作品說明

The F2E 台灣旅遊景點導覽

- 台灣導覽
- 美食住宿
- 景點交通

## 系統說明

Node.js 版本 `v14.18.1`
專案安裝 `npm install`
專案運行 `npm run dev` >> http://localhost:8888
專案打包 `npm run build` >> 資料夾 dist

## 資料夾說明

- dist - 輸出檔案
- node_modules - nodejs 模塊
- src
  - plugins - js 擴充
  - res - 資料資源
  - vue - vue 相關
    - components - 共用元件
      - pages - 頁面元件
        - components - 頁面裡共用元件
